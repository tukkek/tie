Class: Template

Template(selector)

Manages a cloned instance of a template tag. Note that any CSS queries are only performed on the clone sub-tree (other than the one provided in the constructor).

Constructor

new Template(selector)

Finds a template tag, stores it as "template.element" and clones it as "template.root". Note that templates must have a single root element (direct child descendant)!
Parameters:
Name Type Description
selector string A CSS selector. For example: ".name" will select "template.name".
