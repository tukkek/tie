Class: Template

Template(selector, classname)

Manages a cloned instance of a template tag. Note that any CSS queries are only performed on the clone sub-tree (other than the one provided in the constructor).

Constructor

new Template(selector, classname)

Finds a template tag, stores it as "template.element" and clones it as "template.root". Note that templates must have a single root element (direct child descendant)!
Parameters:
Name Type Default Description
selector string A CSS selector. For example: ".name" will select "template.name"
classname string tie For your convenience, this class name is added to all root clone elements. If set to false or an empty string, will not add any class name to the root element.
Properties:
Name Type Description
template element A reference to the template being cloned (will be referred to as the "cloned template").
root element A reference to the root element cloned from the template (wil be referred to as the "template's clone").
