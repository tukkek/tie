Class: Template

Template(selector, classname)

Manages a cloned instance of a template tag. Note that any CSS queries are only performed on the clone sub-tree (other than the one provided in the constructor).

Constructor

new Template(selector, classname)

Finds a template tag, stores it as "template.element" and clones it as "template.root". Note that templates must have a single root element (direct child descendant)! This can be easily achieved by wrapping the template's content in a "div".

Create a new instance for each clone to be managed. Do not re-use the same Template object to "template.create()" multiple clones.
Parameters:
Name Type Default Description
selector string A CSS selector. For example: "#mytemplate" will select "template#mytemplate".
classname string tie For your convenience, this class-name is added to all root clone elements. If set to false, will not add any class-name to the root element.
Properties:
Name Type Description
template element A reference to the template being cloned (will be referred to as the "cloned template").
root element A reference to the root element cloned from the template (referred as the "template's clone").
bindings map A map with arrays of call-back functions by property names (registered with "template.bind()").

bind(property, callback)

Binds a call-back to an object's property. The object must have been previously-trapped with "template.trap()". Any number of properties and call-backs can be registered, including multiple call-backs for a single property.
Parameters:
Name Type Description
property string A property name. For example: a property "p" for a previously-trapped "o" object instance will bind "o.p".
callback function Will be called whenever the bound-property of the previously-trapped-object is changed. This function will be called with the same arguments as: https://mdn.io/handler.set()

set(attribute, value)

Use this to alter a standard HTML attribute (such as an anchor's "href") or to store data in an attribute declared anywhere in your sub-tree. This can be useful in many cases, such as for selecting the element with the CSS query: "*[attribute='value']".

Note that the W3C suggests non-standard attribute names should have the "data-' prefix.

trap(target)

Returns a proxy object with properties that can then be bound with "template.bind()". Bound call-backs will only be called when using this method's returned proxy so it's good practice to discard the original reference.

Each template instance is designed to trap a single object. Consider using more templates or creating Proxy instances in client-code instead of calling this method more than once per instance.
Parameters:
Name Type Default Description
target object false An object instance. If not provided, will default to trapping this template instance.
